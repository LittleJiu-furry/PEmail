PEmail is a simple email server writter by node.js and ts

## Installation

- Clone the repository

```bash
git clone https://github.com/LittleJiu-furry/PEmail.git
```

- install yarn

```bash
npm install -g yarn
```

- install dependencies

```bash
yarn
```

- build the project

```bash
yarn build
```

- run the project

```bash
yarn start
```
or

```bash
node dist/main.js
```
